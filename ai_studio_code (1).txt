# ريفانس المالية - نظام إدارة العقود

تطبيق متكامل لإدارة طلبات الإعفاء والجدولة مع توقيع إلكتروني وتوليد ملفات PDF.

## إعدادات النشر الهامة (Vercel Environment Variables)

لكي يعمل التطبيق بشكل صحيح بعد النشر، يجب إضافة المتغيرات التالية في إعدادات مشروعك على Vercel:

| المتغير | القيمة |
| :--- | :--- |
| `EMAIL_USER` | بريدك الإلكتروني (مثال: rifans.finance@gmail.com) |
| `EMAIL_PASS` | "كلمة مرور التطبيق" من جوجل (App Password) |
| `GEMINI_API_KEY` | مفتاح Gemini API الخاص بك |

## كيفية الحصول على كلمة مرور التطبيق (EMAIL_PASS):
1. انتقل إلى إعدادات حساب Google الخاص بك.
2. ابحث عن "Security" ثم "2-Step Verification".
3. في أسفل الصفحة، اختر "App Passwords".
4. اختر "Mail" و "Other (Custom Name)" وسمّه "Rifans App".
5. انسخ الكود المكون من 16 حرفاً واستخدمه كقيمة لـ `EMAIL_PASS`.
